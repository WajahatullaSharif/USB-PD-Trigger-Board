# **USB C PD Trigger Board**
- The USB PD Trigger Board is based on **AP33771C USB PD3.1 ERP sink controller** from Diodes Incorporated. 
- This board negotiates required **voltages (5V/9V/12V/15V/20V/28V) and current (0 to 5A max)** profiles requested using the respective jumpers with the Type C Power Source (typically charger/adaptor). 
- It includes safety features like Over/Under Voltage, Over Current, Over Temperature Protections along with IN/OUT isolation. The board is also immune to Â± 30 kV Air/Contact ESD dicharge (IEC 61000-4-2)
